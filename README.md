# 📊 Mobile Sales Dashboard using Power BI

An interactive **Power BI dashboard** built to analyze mobile sales performance across multiple cities, brands, customer ratings, and payment methods.

---

## 🚀 Project Overview
This project transforms raw mobile sales data into meaningful business insights using **Power BI, Power Query, and DAX**.  
The dashboard provides a complete overview of:
- Total Sales
- Total Quantity Sold
- Transactions
- Average Sales Value
- Monthly Sales Trends
- City-wise Performance
- Payment Method Analysis
- Customer Ratings

---

## 🛠️ Tools & Technologies Used
- **Power BI**
- **Power Query**
- **DAX**
- **Excel / CSV**
- **Data Visualization**
- **Dashboarding**

---

## 📂 Files Included
- `Mobile_sales_raw.xlsx` → Original raw dataset
- `Mobile_sales_cleaned.csv` → Cleaned and transformed dataset
- `Mobile_sales_dashboard.pbix` → Power BI dashboard file
- `Dashboard_preview.png` → Dashboard screenshot preview

---

## 🧹 Data Cleaning & Transformation
The dataset was cleaned and transformed using **Power Query**:
- Removed null values
- Fixed data types
- Standardized city names
- Cleaned payment mode labels
- Removed duplicate records
- Applied necessary transformations for analysis

---

## 📈 Key Insights
- Identified top-performing cities and brands
- Analyzed monthly sales trends
- Compared customer ratings by model
- Evaluated payment mode usage
- Tracked KPI performance using cards and visuals
